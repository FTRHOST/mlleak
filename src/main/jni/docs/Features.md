# Mod Features Documentation

This document provides an overview of the features implemented in this mod.

## Getting Started
- **[Tutorial & Penjelasan Kode](./Tutorial.md)**: Panduan lengkap cara kerja mod dan pembuatan fitur baru.
- **[Template Fitur](./../feature/Template.h)**: Script contoh yang mudah dipahami untuk memulai.

## Core Components
- **PathManager**: Handles dynamic path resolution for configuration files.
- **Config Polling**: Optimized polling using file modification time (`stat`) to reduce CPU usage.

## List of Features
1. [Unlock Skin](./UnlockSkin.md) - Unlocks all hero skins locally.
